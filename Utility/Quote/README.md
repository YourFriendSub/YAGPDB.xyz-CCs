### Quote
**Usage:**
> To create a quote using your own text or fetch a random quote from internet.

**Trigger**
> Trigger Type: `Command` or `Regex`
> 
> Trigger: `Quote` or `\A(?:\-|<@!?204255221017214977>)\s*(?:quote|quotation|punchline)(?: +|\z)`
