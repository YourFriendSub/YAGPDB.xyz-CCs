### Character Info
**Usage:**
> To get some *basic* information about a unicode character.

**Trigger**
> Trigger Type: `Command` or `Regex`
> 
> Trigger: `CharacterInfo` or `\A(?:\-|<@!?204255221017214977>)\s*(?:char|charinfo|unicode|character|characterinfo|unicodeinfo|uniinfo)(?: +|\z)`
