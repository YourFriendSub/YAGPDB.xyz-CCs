### Automod Helper
**Usage:**
> If you also moderate your server with a mobile phone this is helpful. On mobile, you cannot see how the automod was triggered or which word triggered the inbuilt discord automod. This gives you a embed with all the details about the triggered word, automod name and so on.

**Trigger**
> Trigger Type: `Regex`
> 
> Trigger: `\A` or `.*`
